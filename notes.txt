ORM позволяет из строки БД вернуть обьект и обратно. Это представление обьекта в БД.
К примеру в Хайбернейт есть метод сейв, который подкапотно смапит нужные поля обьекта и вставить в таблицу.

Конфижить можно через xml и java annotation.

1)подключить драйвер нужной БД. Я использовал драйвер неависимый для MySql
2)через Add framework support можно автоматически конфигурировать некоторые файлы

@Entity класс который будет мапится в БД.
POJO - Plain Old Java Object - класс удовлетворяющий ряд условий.
1) приватные поля;
2) гетеры и сетеры;
3) конструктор без аргументов;

в HQL указывать не имя таблицы а имя класса, имя полей.
